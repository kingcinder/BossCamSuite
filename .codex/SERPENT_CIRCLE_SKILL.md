# SERPENT CIRCLE SKILL - BossCamSuite

## Operating Law

Observe live evidence. Verify every claim. Adapt to endpoint/model drift. Persist per-camera truth. Project only proven truth into UI/playback. Re-verify with build/tests.

## Non-Negotiables

- Templates are candidates only.
- Live per-camera proof wins.
- Never infer codec from path suffix.
- Never treat ONVIF declared encoding as playback truth.
- Never treat PTZ service as mechanical PTZ.
- Never fail whole import because one endpoint failed.
- Contradictions become drift notes, not overwritten facts.
- Do not claim live verification unless live probes actually ran.

## Required Final Report

COMPLETED:
- real changes made

VERIFIED:
- commands run and results

PARTIAL:
- implemented but not verified

BLOCKED:
- exact blocker

DRIFT FOUND:
- contradictions found

NEXT RECOVERY STEP:
- one concrete next action
